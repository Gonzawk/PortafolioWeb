Tickets Sales

Este proyecto se diseño para uso particular, busca facilitarle a los clientes una forma rapida y efectiva de obtener entradas para distintos eventos.

Descripción general del proyecto

Tickets Sales es una aplicación web diseñada para gestionar y vender entradas a eventos de manera eficiente y fácil de usar. Este proyecto fue desarrollado utilizando tecnologías modernas como ASP.NET Core para el backend, donde se crean APIs RESTful que se comunican con una base de datos SQL Server para almacenar y gestionar la información de los eventos, los usuarios y las ventas de entradas.

Para garantizar la seguridad de los usuarios, Tickets Sales implementa BCrypt para el cifrado seguro de contraseñas y JWT (JSON Web Tokens) para el manejo de autenticación y autorización, asegurando un acceso controlado y eficiente a las funcionalidades de la aplicación.
